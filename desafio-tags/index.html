Tags <H1> A <H1>

<H1> a <H6> são tags de cabeçalho, usadas para definir títulos e subtítulos em um documento HTML.

Tag <p>

A tag <p> define um parágrafo de texto. É usada para agrupar blocos de texto em um documento HTML.

Tag <mark>

A tag <mark> é usada para destacar ou marcar texto, geralmente com um fundo amarelo, como se estivesse marcado com um marcador de texto.

Tag <small>

A tag <small> é usada para diminuir o tamanho da fonte do texto contido nela, muitas vezes usada para notas de rodapé ou detalhes menos importantes.

Tag <i>

A tag <i> aplica estilo itálico ao texto. Geralmente é usada para indicar um texto que deve ser lido com ênfase ou como se fosse um termo técnico.

Tag <u>

A tag <u> sublinha o texto. Tradicionalmente usada para indicar hiperlinks, embora seu uso hoje em dia seja mais estilístico.

Tag <strong>

A tag <strong> indica que o texto tem uma importância forte, estilizando-o em negrito para dar ênfase.

Tag <ol>

A tag <ol> cria uma lista ordenada (numerada), onde cada item é precedido por um número.

Tag <ul>

A tag <ul> cria uma lista não ordenada (com marcadores), onde cada item é precedido por um ponto ou marcador.

Tag <li>

A tag <li> define um item em uma lista, sendo usada dentro das tags <ol> e <ul> para criar itens de lista.

Tag <a>

A tag <a> define um hiperlink, que pode ser usado para vincular a outra página da web, recurso, ou qualquer outro destino de URL.

Tag <hr>

A tag <hr> insere uma linha horizontal, usada para separar conteúdos em uma página web.

Tag <sub>

A tag <sub> é usada para texto em subscrito, onde o texto é exibido abaixo da linha base (por exemplo, fórmulas químicas).

Tag <sup>

A tag <sup> é usada para texto em sobrescrito, onde o texto é exibido acima da linha base (por exemplo, expoentes matemáticos).

Tag <blockquote>

A tag <blockquote> é usada para indicar uma citação em bloco, destacando um texto de uma fonte diferente, geralmente com uma indentação.
